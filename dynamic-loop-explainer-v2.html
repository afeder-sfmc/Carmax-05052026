How it works — step by step

1

Email Sent via MCA

Each vehicle card includes a tracked link containing the vehicle ID as a URL parameter, rendered via Handlebars JS from a Data Graph.

2

Customer Clicks ❤️

The click resolves to a Data Cloud Streaming Ingestion API endpoint, passing the subscriber key and vehicle ID in real time.

3

Profile Updated

Data Cloud writes a new engagement record — "favorited vehicle" — on to the customer's Unified Individual profile instantly.

4

Platform Reacts

Segments refresh, flows trigger, calculated insights update, and CRM surfaces the signal — all on the Salesforce platform, no custom dev.
