Storyline

CarMax marketers launch a campaign for customers with expiring rewards value and low recent purchase activity. They use MCA Campaign Agent to draft quickly, Data Cloud segmentation and Calculated Insights to target precisely, dynamic email loops to personalize at scale, and MCA Flow to orchestrate cross-channel engagement with controls.

Keep this positioning line consistent: Data Cloud is the decisioning context for MCA targeting, not the center of the demo.

Harbor Cruise Run of Show

Dock: Business Framing
3 min
  • Set objective: conversion + engagement + message pressure control.
  • Explain no-slide product-first walkthrough.
Stop 1: Campaign Agent
8 min
  • Draft from natural-language campaign brief.
  • Confirm audience intent and channel strategy.
Stop 2: Segment Creation
8 min
  • Create with Einstein, then refine predicates.
  • Use loyalty, recency, value, and eligibility logic.
Stop 3: CI + Engagement Overlay
6 min
  • Add Calculated Insights to improve audience quality.
  • Apply engagement/frequency suppression controls.
Stop 4: Dynamic Loop Email
8 min
  • Show repeating blocks for personalized offers.
  • Preview 2-3 personas with different outcomes.
Stop 5: MCA Flow Orchestration
7 min
  • Show entry, branch logic, wait steps, and fallback path.
  • Highlight controls for send pressure and exclusions.
Stop 6: Reporting
3 min
  • Open campaign/journey performance view.
  • Explain how outcomes feed next segmentation cycle.
Wrap: Value Recap
2 min
  • Restate full loop: brief -> audience -> execution -> learning.
  • Anchor to marketer productivity and governance.

Use Cases in the Demo

  • Primary: Expiring rewards credits re-engagement with service-ready offer.
  • Feature moment: Dynamic loop email showing multiple personalized modules.
  • Governance moment: Engagement/frequency suppression to avoid over-messaging.
  • Use business language first, platform language second.
  • When showing Data Cloud, tie each field to one send decision.
  • Do not deep-dive ingestion architecture unless asked.
  • Close each stop with the same transition phrase: "Now that we have this, here is the next marketer decision."

Data Cloud Scope (Deliberately Narrow)

Show This

  • Segment attributes needed for campaign qualification.
  • Calculated Insights: value, recency, and risk proxy.
  • Engagement signals for suppression and prioritization.

Avoid This

  • Long architecture detours.
  • Broad CDP displacement narratives.
  • Low-value setup minutiae during live exec flow.

Risks & Mitigations

  • Generation lag: keep prebuilt campaign and segment backups open.
  • Sparse reporting data: use prepared historical view and expected learning-loop talk track.
  • Scope drift into deep Data Cloud: time-box to decision-critical fields.
